# Data Usage Disclosure — ProjectCare by iCareNOW

ProjectCare by iCareNOW is provided by iCareNOW (icarenow.io).

When you use this GPT:
- Your task estimates (optimistic, most likely, pessimistic values) are sent to the ProjectCare by iCareNOW API to run probability calculations.
- Your API key is transmitted with each request solely to authenticate and track credit usage.
- Your email address is collected only if you request a free trial key, and is used to deliver that key.
- Task estimates and results are processed in memory and are not automatically stored. If you use the **Save Session** feature, your task inputs and results summary are stored server-side (linked to your email) so you can reload them later. You may request deletion of saved sessions by contacting icarenow.io.
- iCareNOW does not sell or share your data with third parties.
